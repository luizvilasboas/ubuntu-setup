# Easy Arch

> A simple and easy way to install all the apps you need on your Arch Linux distribution.

## About the Project

This project is a collection of scripts to automate the installation of various applications on an Arch-based Linux distribution. It provides a simple way to set up a new machine with all the necessary software, without having to manually install each application one by one.

## Tech Stack

The main technologies and libraries used in this project are:

* **Ruby**
* **Shell Script**

## Usage

Follow the instructions below to set up and run the project locally.

### Prerequisites

You need to have the following software installed on your machine:

* [Ruby](https://www.ruby-lang.org/en/documentation/installation/)
* [Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)

### Installation and setup

1.  **Clone the repository**
    ```bash
    git clone https://github.com/luizvilasboas/easy-arch.git
    ```

2.  **Navigate to the project directory**
    ```bash
    cd easy-arch
    ```

3.  **Install dependencies**
    ```bash
    bundle install
    ```

### Running the Project

To start the application, run the command below:

```bash
sudo ./main.sh
````
